Định nghĩa: HTML: Hypertext Markup Language - Ngôn ngữ đánh dấu siêu văn bản

Các thẻ cơ bản:

  • Thẻ p (paragraph):
    • <p> content </p>
  • Thẻ h (heading):
    • <h1 id="Heading1">

      H1 is the biggest of heading tags with ID = Heading1

      </h1>
    • <h2>

      H2 is only smaller than h1 on heading tags

      </h2>
    • ...
    • <h6>
      H6 is the smallest of heading tags </h2>
  • Mặc định: tất cả space hoặc xuống dòng sẽ bị mất đi khi hiển thị trên trình duyệt
    • Dùng   để định nghĩa một dấu cách
    • Dùng <br> (break line) để định nghĩa một dấu xuống dòng
    • Dùng <hr> (horizontal rule) để định nghĩa một dòng kẻ ngang
  • Định dạng:
    • <b> Thẻ b (bold) make this text bold </b>
    • <strong> Thẻ strong make this text bold (mang cả về mặt nghĩa nhấn mạnh) </strong>
    • <i> Thẻ i (italic) make this text italic </i>
    • <em> Thẻ em (emphasize - nhấn mạnh) make this text italic (mang cả về mặt nghĩa nhấn mạnh) </strong>
    • <small> Thẻ small make this text small </small>
    • <mark> Thẻ small make this text is marked </mark>
    • <del> Thẻ small make this text is deleted </del>
    • <ins> Thẻ ins (inserted) make this text is inserted </ins>
    • <sub> Thẻ sub (subscript) make this text like subscript </sub>
    • <sup> Thẻ sup (superscript) make this text like superscript </sup>
  • Comments: Là một công cụ dùng cho con người dễ dàng đọc hiểu và maintain code
    • <!-- Đoạn văn bản này sẽ được browser bỏ qua -->
  • Images:
    • <img> src="./images/workplace.jpg" alt="workplace.jpg" </b>
    • <img> src="./images/workplace.jpg" alt="workplace.jpg" usemap="#workmap" </b> <map name="workmap"> <area name="workmap" shape="rect|circle" coords="x,x,x,x" href="" alt=""> </area> </map>
    • Background Image: (css) body { background-image: url(""); }
  • Links:
    • Local: <a href="../index.html">Click me to go file with href="../index.html" </a>
    • External: <a href="https://google.com">Click me to go google </a>
    • Target:
      • <a href="https://google.com" target="_self">Click me to go google open in the same tab </a>
      • <a href="https://google.com" target="_blank">Click me to go google open in another tab </a>
    • Button as Link: <button onclick="document.location = '../index.html'" > Click me </button>
    • Image as Link: <a href="https://google.com" target="_blank"> <img src="./images/programmer.jpeg" alt="programmer" width="50px"> </img> </a>
    • Link Color:
      • a:link { color: green}
      • a:hover { color: red}
      • a:visited { color: pink }
      • a:active { color: yellow }
      <a href="http://google.com" target="_self">Click me! </a>
    • Link bookmarks:
      • <a href="#heading1">Click me to h1 </a>
      • <a href="index.html#heading1">Click me to h1 </a>
  • Table:
    • <table>
      table tag <tr>
      table row <td>Id</td>
      table column css: table, tr, td border-collapse: collapse;
      Id Name
      1 User name
    • Attribute: colspan, rowspan
    • Caption: <caption> ngay sau table
  • Lists: This is a example of unorder list
    • Unorder: ul
    • Order: ol
    • Item: li
    • Choose attribute "type": A, a, ....
