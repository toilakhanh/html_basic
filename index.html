Một số thẻ html đã học

  • Thẻ h (heading):
    • <h1 id="Heading1">

      H1 is the biggest of heading tags with ID = Heading1

      </h1>
    • <h2>

      H2 is only smaller than h1 on heading tags

      </h2>
    • ...
    • <h6>
      H6 is the smallest of heading tags
  • Mặc định: tất cả space hoặc xuống dòng sẽ bị mất đi khi hiển thị trên trình duyệt
    • Dùng   để định nghĩa một dấu cách
    • Dùng <br> (break line) để định nghĩa một dấu xuống dòng
  • Định dạng:
    • <b> Thẻ b (bold) make this text bold </b>
    • <strong> Thẻ strong make this text bold (mang cả về mặt nghĩa nhấn mạnh) </strong>
    • <i> Thẻ i (italic) make this text italic </i>
    • <em> Thẻ em (emphasize - nhấn mạnh) make this text italic (mang cả về mặt nghĩa nhấn mạnh) </em>
    • <small> Thẻ small make this text small </small>
    • <mark> Thẻ small make this text is marked </mark>
    • <del> Thẻ small make this text is deleted </del>
    • <u> Thẻ u (underline) make this text underline </u>
    • <sub> Thẻ sub (subscript) make this text like subscript </sub>
    • <sup> Thẻ sup (superscript) make this text like superscript </sup>
  • Comments: Là một công cụ dùng cho con người dễ dàng đọc hiểu và maintain code
    • <!-- Đoạn văn bản này sẽ được browser bỏ qua -->
  • Images:
    • <img> src="url image" alt="redpanda.jpg" </b>
    • Background Image: (css) body { background-image: url(""); }
  • Table:
    • <table>
      table tag <tr>
      table row <td>Id</td>
      table column
      Id Name
      1 User name
    • Attribute: colspan, rowspan
    • Caption: <caption> ngay sau table
  • Lists: This is a example of unorder list
    • Unorder: ul
    • Order: ol
    • Item: li
    • Choose attribute "type": A, a, ....
  • Blocks and Inline:
    • Blocks: <address><article><aside><blockquote><canvas><dd><div><dl><dt><fieldset><figcaption><figure><footer><form><h1>-<h6><header><hr><li><main><nav><noscript><ol><p><pre><section><table><tfoot><ul><video></video>
    • Inline: <a><abbr><acronym><b><bdo><big><br><button><cite><code><dfn><em><i><img><input><kbd><label><map><object><output><q><samp><script><select><small><span><strong><sub><sup><textarea><time><tt><var>
